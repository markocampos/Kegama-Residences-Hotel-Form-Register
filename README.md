## System Access

The Management System is protected and accessible only via the authorized management URL.

- **Login Portal:** `/mgmt/login/` (Contact Admin for specific URL)

## Core Functions

### 1. Guest Registration
- **Role:** Front Desk / Guest
- **Function:** Digital form replacement with validation.
- **Privacy:** Data is stored securely.

### 2. Management Dashboard
- **Live Monitor:** View real-time guest submissions.
- **Management:** Assign rooms, apply discounts, and manage requests.
- **Printing:** Generate official PDF Registration Forms with "Payment Slip" automatically.

### 3. Analytics & Financials
- **Revenue Tracking:** Real-time view of daily income.
- **Data Visualization:** Charts for booking sources and revenue trends.

### 4. Security Settings
- **Maintenance Mode:** Ability to lock the guest form instantly.
- **Access Code:** Restrict form access to authorized users only.
- **PIN Management:** Securely rotate the Admin PIN from the dashboard.
